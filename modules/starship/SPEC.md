# Starship

Enables the Starship cross-shell prompt with integrations for bash, fish, and nushell.

## Usage

```nix
starship.enable = true;
```

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| enable | bool | false | Enable starship prompt |

## Notes

- Automatically enables integration with bash, fish, and nushell.
